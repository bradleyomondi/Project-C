FIFA 26 Inspired Football Game MVP

Build a football/soccer game MVP inspired by FIFA 26 with modern gameplay, online features, and realistic presentation. The goal is to create a playable vertical slice, not a full AAA game.

Core MVP Requirements

  • 3D third-person football gameplay
  • 11v11 matches with AI-controlled teammates
  • Basic player movement:
    • Sprint
    • Pass
    • Shoot
    • Tackle
    • Cross
    • Through-ball
  • Ball physics with realistic momentum and collisions
  • Broadcast-style FIFA camera system
  • Player stamina system
  • Simple referee logic:
    • Fouls
    • Offsides
    • Yellow / Red cards
  • Scoreboard, timer, and match UI
  • Team selection screen
  • 4 playable teams with unique kits and ratings
  • Basic commentary placeholders
  • Crowd ambience and stadium lighting
  • Single-player vs AI
  • Local multiplayer support
  • Replay highlights

AI Systems

  • Teammate positioning AI
  • Goalkeeper AI
  • Tactical formations:
    • 4-3-3
    • 4-4-2
    • 3-5-2
  • Difficulty Levels:
    • Easy
    • Medium
    • Legendary

Game Modes

  • Quick Match
  • Penalty Shootout
  • Tournament Bracket

Visual Direction

  • Realistic graphics inspired by EA Sports FC / FIFA 26
  • Dynamic weather:
    • Rain
    • Sunset
    • Night matches
  • Motion-captured animation style
  • Smooth transitions and cinematic intros

Technical Stack

  • Engine: Unreal Engine 5 or Unity HDRP
  • Multiplayer-ready architecture
  • Modular player controller system
  • Scriptable team/player database
  • Controller + keyboard support

Monetization Placeholder Systems

  • Ultimate Team-style card framework (mock only)
  • Battle pass UI placeholder
  • Cosmetic unlock system

Deliverables

  • Playable prototype
  • Main menu UI
  • One polished stadium
  • Four complete teams
  • Match flow from kickoff to final whistle
  • Optimized build for PC

Stretch Goals

  • Online matchmaking
  • Career mode framework
  • Player transfer market
  • Dynamic commentary AI
  • Advanced skill moves
  • VAR replay system

Design Philosophy

Focus On:

  • Responsive gameplay
  • Realistic football feel
  • Polished controls
  • Replayability
  • Scalable architecture

Avoid:

  • Massive content scope
  • Licensing complexity
  • Hundreds of teams initially
